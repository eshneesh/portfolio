Что поставить на страницу Power BI

структура листа

Срезы сверху

Дата изменения, тип заявки, подразделение, исполнитель, текущий статус. Срез статуса лучше брать из StatusSteps.

KPI HTML

Один HTML Content visual с мерой KPI Strip HTML: всего, в работе, средний цикл, нарушения SLA.

Воронка статусов

HTML visual Current Status Funnel HTML или обычная bar chart: StatusSteps[Status] + Current Status Requests.

Timeline заявки

HTML visual Selected Request Timeline HTML. Работает после выбора одной заявки в таблице или через drill-through.

Узкие места

Bottleneck Heatmap HTML или матрица: StatusSteps[Status], Avg Stage Hours, Median Stage Hours, SlaHours.

Проблемные заявки

Таблица с RequestID, текущим статусом, возрастом текущего этапа, циклом, признаком SLA.

Минимальные меры

полная версия в .txt
Requests = DISTINCTCOUNT ( 'StatusLog'[RequestID] )

Active Requests = [Requests] - [Completed Requests]

Avg Stage Hours = AVERAGE ( 'StatusLog'[StageDurationHours] )

Avg Cycle Hours =
AVERAGEX (
    VALUES ( 'StatusLog'[RequestID] ),
    VAR StartAt = CALCULATE ( MIN ( 'StatusLog'[ChangedAt] ) )
    VAR LastAt = CALCULATE ( MAX ( 'StatusLog'[ChangedAt] ) )
    RETURN DIVIDE ( DATEDIFF ( StartAt, LastAt, MINUTE ), 60 )
)

Current Status Requests =
SUMX (
    VALUES ( 'StatusLog'[RequestID] ),
    VAR CurrentRequest = 'StatusLog'[RequestID]
    VAR LastRow =
        TOPN (
            1,
            FILTER ( ALL ( 'StatusLog' ), 'StatusLog'[RequestID] = CurrentRequest ),
            'StatusLog'[ChangedAt], DESC
        )
    RETURN IF ( MAXX ( LastRow, 'StatusLog'[Status] ) = SELECTEDVALUE ( 'StatusSteps'[Status] ), 1, 0 )
)

HTML-визуалы в DAX

готовые меры в файле

KPI Strip HTML

Горизонтальная строка карточек. Вставляется в HTML Content visual во всю ширину страницы.

Current Status Funnel HTML

Показывает распределение заявок по последнему статусу, а не количество событий по статусам.

Selected Request Timeline HTML

Рисует хронологию выбранной заявки, длительность каждого этапа и подсветку превышения SLA.

Bottleneck Heatmap HTML

Подсвечивает этапы с максимальным средним временем прохождения.

Обычные visuals

Линейный график Events по ChangedAt, таблица проблемных заявок, drill-through страница заявки.

Важно

HTML visual должен поддерживать inline CSS. JavaScript не нужен: весь HTML генерируется DAX-мерой.
